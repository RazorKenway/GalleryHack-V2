## INSTALLATION [Termux] [Linux] :

* `apt-get update -y`
* `apt-get upgrade -y`
* `apt install git -y`
* `apt install python -y`
* `git clone https://github.com/RazorKenway/GalleryHack-V2.git`
* `cd GalleryHack-V2`
* `cd Termux`
* `python setup.py (only once)`
* `python main.py`
